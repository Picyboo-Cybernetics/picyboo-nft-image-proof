# Beispielablauf

Dieser Ordner zeigt, wie sich aus einem Bild eine strukturierte Beschreibung erzeugen lässt.

## Schritte
1. Abhängigkeiten installieren (`npm install`).
2. Skript ausführen:
   ```bash
   node examples/proof-demo.mjs
   ```
3. Ergebnis unter `examples/output/sample-proof.json` prüfen.

Die JSON-Datei enthält Hashes, Metadaten und den CTX-Token des Beispielbilds.
